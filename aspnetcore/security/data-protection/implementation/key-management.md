## <a name="default-key-selection"></a><span data-ttu-id="66635-116">Sélection de la clé par défaut</span><span class="sxs-lookup"><span data-stu-id="66635-116">Default key selection</span></span>

<span data-ttu-id="66635-123">Il existe une exception.</span><span class="sxs-lookup"><span data-stu-id="66635-123">There is an exception.</span></span> <span data-ttu-id="66635-124">Si le développeur
