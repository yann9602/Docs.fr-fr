Ce didacticiel décrit ASP.NET Core MVC avec des contrôleurs et des vues. Les pages Razor sont une nouvelle alternative dans ASP.NET Core 2.0. Il s’agit d’un modèle de programmation basé sur les pages qui permet de créer une interface utilisateur web de manière plus facile et plus efficace. Nous vous recommandons de suivre le didacticiel sur les [pages Razor](xref:mvc/razor-pages/index) avant la version MVC. Le didacticiel sur les pages Razor :

* est plus facile à suivre ;
* couvre davantage de fonctionnalités ;
* représente la meilleure approche pour le développement de nouvelles applications.
