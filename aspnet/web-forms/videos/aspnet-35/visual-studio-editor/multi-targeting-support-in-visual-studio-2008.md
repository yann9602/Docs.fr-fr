<a name="multi-targeting-support-in-visual-studio-2008"></a>Plusieurs ciblant la prise en charge dans Visual Studio 2008
====================
par [Joe Stagner](https://github.com/JoeStagner)

Je vais illustrer la fonctionnalité MultiTarget de Visual Studio 2008. Dans le passé, vous deviez choisir la version de Visual Studio pour correspondre à la version ciblée de .NET. Par exemple, Visual Studio 2005 a travaillé avec .NET 2.5. Désormais, avec Visual Studio 2008, vous pouvez choisir la version de .NET approprié pour votre application.
